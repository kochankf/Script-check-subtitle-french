Script de check de sous-titre Français .ass 


Prérequis :  [LM Studio](https://lmstudio.ai/download), [Gemma 4](https://huggingface.co/google/gemma-4-E4B-it) meilleur résultat avec ce modèle, et si vous avez pas, intallé requests : pip install requests

Instalation et fonctionnement : 
1. Mets tout au même endroit

Dans un dossier (par exemple sur le Bureau), mets :
ton script → corrige.py
ton fichier → xxxx.ass

2. Ouvre un terminal dans ce dossier
- Méthode rapide :
Va dans ton dossier
Clique dans la barre d’adresse (en haut)
Tape : cmd
Appuie sur Entrée
👉 Ça ouvre une fenêtre noire (invite de commande)

3. Lance le script
Dans la fenêtre, tape :

python corrige.py "xxxx.ass"

puis Entrée

👉 Résultat

Ca fait défiler les lignes + suggestion [].
Avec une sortie Backup et une corrigé.

Attention : pensez à bien relir derrière !
